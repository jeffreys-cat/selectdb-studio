description: 欢迎使用 SelectDB Studio。
---

SelectDB Studio 是基于原来云/企业版 WebUI 发展而来的桌面版客户端，支持 MacOS / Windows 等桌面操作系统。

用户可以直接下载安装使用桌面客户端而不需要 Linux 服务器部署服务/升级版本，降低用户的使用/部署成本，触达更多用户。我们也可以把更多功能给用户使用接收反馈，促进商业产品的功能稳定可靠。

桌面客户端所有功能和 WebUI 保持一致，但增加了多连接/SSH Tunnel登录集群，支持 Apache Doris 和 SelectDB。
